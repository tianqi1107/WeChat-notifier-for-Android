# 微信通知提醒 (WeChat Notification Notifier)

一款 Android 应用，可以在手机静音模式下，为你特别关心的微信联系人触发震动提醒。

## 功能特性

- **通知监听**：通过 Android NotificationListenerService 监听微信消息通知
- **震动提醒**：特别关心联系人的消息可以触发震动，即使手机处于静音或勿扰模式
- **联系人白名单**：从微信通知历史中自动识别联系人，勾选加入特别关心列表
- **时间段设置**：可设置功能生效的时间段（如仅在夜间 22:00-07:00 生效）
- **开机自启**：手机重启后自动恢复监听服务
- **后台保活**：前台服务确保应用不被系统杀死

## 安装方法

### 方式一：直接安装 APK

1. 从 [Releases](https://github.com/tianqi1107/WeChat-notifier-for-Android/releases) 下载最新的 APK
2. 在手机上打开 APK 文件安装
3. 如果提示"未知来源应用"，请允许安装

### 方式二：从源码构建

```bash
# 克隆仓库
git clone https://github.com/tianqi1107/WeChat-notifier-for-Android.git
cd WeChat-notifier-for-Android

# 确保已安装 Android SDK（API 35）和 JDK 17+
# 构建 Debug APK
gradlew.bat assembleDebug

# APK 输出位置
# app/build/outputs/apk/debug/app-debug.apk
```

## 使用说明

### 第一步：授予通知监听权限

1. 打开应用
2. 点击"前往设置"按钮
3. 在系统设置中找到"微信通知提醒"并开启通知访问权限
4. 返回应用

### 第二步：开启监听

1. 打开"启用监听"开关
2. 应用将开始在后台监听微信通知

### 第三步：管理特别关心联系人

1. 点击"管理特别关心联系人"
2. 让朋友给你发几条微信消息，联系人会自动出现在列表中
3. 勾选你想特别关心的人

### 第四步：设置时间段（可选）

1. 开启"仅在指定时间段内生效"
2. 设置开始时间和结束时间
3. 只有在该时间段内收到的消息才会触发提醒

### 第五步：电池优化设置（重要）

为确保应用在后台持续运行，请进行以下设置：

1. 点击"前往电池设置"
2. 找到"微信通知提醒"
3. 关闭电池优化 / 允许后台运行
4. 在 iQOO 手机上，还需要在 iManager 中允许自启动

## 注意事项

- 需要 Android 8.0 (API 26) 及以上版本
- 首次使用必须授予通知监听权限
- iQOO/vivo 手机需要手动关闭电池优化并允许自启动
- 微信通知格式可能随版本更新变化，如遇问题请反馈
- 本应用不会读取或存储任何聊天内容，仅识别发送者昵称
